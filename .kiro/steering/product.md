# Product

Personal Dashboard — single self-contained HTML file.

Four widgets on purple-to-blue gradient background (`#6c63ff` → `#48b5f3`):
- **Clock & Greeting** — live time/date, context-aware greeting
- **Focus Timer** — 25-min countdown with start/stop/reset
- **To-Do List** — add/edit/delete/complete tasks
- **Quick Links** — save and launch URLs in new tab

All user data persisted in `localStorage`. Works fully offline after first load. No server, no backend.
