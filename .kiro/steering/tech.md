# Tech Stack

**Vanilla HTML + CSS + JS only.** No frameworks, no libraries, no build tools.

- HTML5, CSS3, vanilla JavaScript (ES6+)
- Google Fonts (Inter) — only external dependency
- `localStorage` for all persistence
- No npm, no bundler, no transpiler

## Constraints

- All CSS and JS must be **inline** in the single `.html` file
- No external JS/CSS files
- No third-party frameworks (React, Vue, jQuery, etc.)

## Commands

No build step. Open the `.html` file directly in a browser.
